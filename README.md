# TKO-Postbox.

![TKO-Postbox](https://repository-images.githubusercontent.com/232512685/70107a80-31f8-11ea-9436-66c4dc25c0e2)

Minimalist repo to experiment with knockout-postbox and TKO.

## Installation

Install dependencies:
```
npm i
```
Serve project:
```
npm run start
```
